thundr-contrib-freemarker FreeMarker

A thundr module for rendering views using FreeMarker.

FreeMarker is a templating language. Once this module has been included and configured, you can return a FreemarkerView from any controller method to render content to the request client.

Setting up

Add the module dependency to your build tool and require the module in your ApplicationModule:

pom.xml - Click here to find the latest version

{% highlight xml %} ... com.atomicleopard.thundr thundr-contrib-freemarker 2.0.0-alpha compile ... {% endhighlight %}

Your ApplicationModule

{% highlight java %} ... @Override public void requires(DependencyRegistry dependencyRegistry) { super.requires(dependencyRegistry); dependencyRegistry.addDependency(FreemarkerModule.class); } ... {% endhighlight %}

FreemarkerView

Once set up, you can return a FreemarkerView from a controller method to render output to the request client. FreemarkerView can also be used when sending emails.

{% highlight java %} ... public FreemarkerView view() { Map model = model(); populateReferenceData(model); return new FreemarkerView("view.ftl", model); } ... {% endhighlight %}

FreemarkerView is a template view, so you can set headers, cookies, status code etc on it.

It also respects the Global Model.
