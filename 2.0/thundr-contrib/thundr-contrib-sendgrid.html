thundr-contrib-sendgrid SendGrid

Overview

SendGrid is a platform for sending, receiving and tracking email and email campaigns.

thundr-contrib-sendgrid provides integration of SendGrid using the standard thundr Mailer API

Source on github

Setting Up

To use thundr-contrib-sendgrid add the library to your application, add a dependency on the SendGridModule and configure your SendGrid api key using your application.properties, or in code.

Including the thundr-contrib-sendgrid jar in your project will enable you to instantiate a SendGridMailer, which sends email using the standard thundr Mailer, so you can swap it out interchangably with the built in JavaMailMailer.

Module Features

Once you've configured the SendGridModule you'll be able to send email using the Mailer API. {% highlight java %} ... private Mailer mailer; public MyService(Mailer mailer){ this.mailer = mailer; } public void sendAnEmail(){ // This mail will be sent using SendGrid's infrastructure. mailer.mailer() .from("from@email.com", "From") .to("recipient@email.com", "Recipient") .subject("subject") .body(new StringView("body")) .send(); } ... {% endhighlight %}
