thundr-contrib-docraptor Docraptor

A thundr module for rendering documents using Docraptor. Source on Github

Setting up

Add the module dependency to your build tool and require the module in your ApplicationModule:

pom.xml - Click here to find the latest version

{% highlight xml %} ... com.atomicleopard thundr-contrib-docraptor 1.0.0 compile ... {% endhighlight %}

Your ApplicationModule

{% highlight java %} ... @Override public void requires(DependencyRegistry dependencyRegistry) { super.requires(dependencyRegistry); dependencyRegistry.addDependency(DocraptorModule.class); } ... {% endhighlight %}

Your application.properties

{% highlight properties %} ... docraptorApiKey=YOUR_KEY_HERE ... {% endhighlight %}

Docraptor

The Docraptor interface will then be available for injection into services and controllers. Draw a PDF by making a render call passing in a DocraptorRequest with the appropriate fields.

{% highlight java %} private Docraptor docraptor; ... DocraptorRequest request = DocraptorRequest.ForUrl("filename.pdf", absoluteUrl) .withJavascript(true) .withTestMode(testMode); byte[] pdfData = docraptor.render(request); ... {% endhighlight %}

For more detail, see the docraptor API docs
