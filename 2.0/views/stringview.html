StringView

StringView is a simple view which returns the given string as the response content, defaulting to a response code of 200 OK and a content type of text/plain.

StringView is intrinsic, built into thundr core.

{% highlight java %} ... return new StringView("Hello!"); ... {% endhighlight %}

StringView is a base view so you can set content type, cookies, status code etc as usual.

{% highlight java %} ... return new StringView("Hello!") .withStatus(StatusCode.BadRequest) .withHeader("X-App-Specific", "special-value") .withContentType(ContentType.TextPlain) .withCookie("user", userId); ... {% endhighlight %}
