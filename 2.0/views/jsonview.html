thundr JsonView

JsonView

JsonView is a data view which uses the JsonViewResolver to render the given model object to json.

JsonView uses the Gson library to serialize the given object to json, and return it to the request client.

JsonView is intrinsic, built into thundr core.

{% highlight java %} ... MyDto dto = ... return new JsonView(dto); ... {% endhighlight %}

You can override the GsonBuilder used by the JsonViewResolver by creating and registering your own instance at startup.

{% highlight java %} ... GsonBuilder gsonBuilder = GsonSupport.createBasicGsonBuilder(); gsonBuilder.registerTypeAdapter(Money.class, new MoneyTypeConverter()); ViewResolverRegistry viewResolverRegistry = injectionContext.get(ViewResolverRegistry.class); viewResolverRegistry.addResolver(JsonView.class, new JsonViewResolver(gsonBuilder)); ... {% endhighlight %}

JsonpView

JsonpView is a data view which uses the JsonpViewResolver to render the given model object to jsonp.

JsonpView uses the Gson library to serialize the given object to jsonp, and return it to the request client.

JsonpView is intrinsic, built into thundr core.

It operates the same way as the JsonView and JsonViewResolver

{% highlight java %} ... MyDto dto = ... return new JsonpView(dto); ... {% endhighlight %}

You can override the GsonBuilder used by the JsonpViewResolver the same way as the JsonViewResolver.
