FileView

The FileView streams the data model to the request client with the specified content-type, and filename and disposition (as content-disposition).

The FileViewResolver is responsible for serving the FileView. FileView extends base view, you can set content type, status, headers, cookies etc.

FileView is intrinsic, built into thundr core.

{% highlight java %} public FileView get(){ ... return new FileView(filename, data, "text/plain").withDisposition(Disposition.Attachment); } {% endhighlight %}
