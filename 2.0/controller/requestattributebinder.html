thundr Binding request attributes

thundr provides the RequestAttributeBinder to bind request attributes onto controller parameters.

Request attributes are attributes on the servlet request. These are typically added through servlet filters, thundr filters, thundr interceptors or by the container itself.

For example, if you have a request attribute of type com.company.RequestId named 'requestId', the following would bind:

{% highlight java %} public View method(RequestId requestId){ ... } {% endhighlight %}
