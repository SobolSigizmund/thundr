thundr Binding path variables

thundr provides the PathVariableBinder to bind path variables onto controller parameters.

Path variables are the named variables as portion of the route. e.g.

{% highlight java %} routes.add(GET, "/{path}/{variable}", Controller.class, "method"); {% endhighlight %}

will bind to

{% highlight java %} public View method(long path, String variable){ ... } {% endhighlight %}

This binder uses the TransformerManager, and so can bind to any target type which can be converted from a String.
