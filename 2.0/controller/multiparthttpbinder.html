thundr Binding Multipart Http requests

thundr provides the MultipartHttpBinder to bind form parameters when the request has the content type multipart/form-data

Binding of each parameter occurs using either a ParameterBinder or a BinaryParameterBinder as appropriate.

{% endhighlight %}

{% highlight java %} public JspView postFile(String name, MultipartFile fileData){ ... } {% endhighlight %}

as well as the following two variants

{% highlight java %} public JspView postFile(String name, byte[] fileData){ ... } {% endhighlight %} {% highlight java %} public JspView postFile(String name, InputStream fileData){ ... } {% endhighlight %}

To extend the types that can be bound to, you can register custom BinaryParameterBinder implementations with the ParameterBinderRegistry at startup.
