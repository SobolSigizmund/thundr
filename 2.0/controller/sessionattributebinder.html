thundr Binding session attributes

thundr provides the SessionAttributeBinder to bind session attributes onto controller parameters.

Session attributes are attributes on the session which can be added through filters or by the container itself

For example, if you have a session attribute of type com.company.User named 'user', the following would bind:

{% highlight java %} public View method(User user){ ... } {% endhighlight %}
