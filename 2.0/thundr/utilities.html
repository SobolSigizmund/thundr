thundr utilities

thundr has a lot of utility code which can help you succinctly and rapidly deliver your application code.

com.threewks.thundr.exception.BaseException

A base exception class the allows you to pass a string format and args instead of just a message. Most if not all of the exceptions from thundr extend this - and you can too.

com.threewks.thundr.util.Encoder

Encoder provides a fluent api for encoding strings and binary data in various ways.

If you need to do something with Hex, Base32, Base64, MD5, SHA1, SHA256 etc, then this is a good place to look.

{% highlight java %} // An example of base64 encoding a username and password String basicAuth = "Basic " + new Encoder(username + ":" + password).base64().string(); // MD5 a byte array String hexMd5 = new Encoder(byteArray).md5().hex().string(); {% endhighlight %}

com.threewks.thundr.configuration.Environment

This class contains the current environment as configured for thundr.

This is useful if you need to switch implementations or behaviour based on the current environment.

Generally, this sort of logic only belongs in your ApplicationModule, where you would inject a different implementation based on the environment.

com.threewks.thundr.logger.Logger

A statically accessible Logger which sits on top of Java Util Logging (JUL).

All of thundr logs through this interface, and you can too.

Log methods receive a string format and varargs parameters. The output string will not be processed unless the appropriate level of logging is enabled, this allows you to avoid wrapping logging calls in if conditions testing for the current log level.

The formatting is the same as String.format(), as defined here.

{% highlight java %} Logger.info("Some formattable message %d: %s", decimal, string); {% endhighlight %}

com.threewks.thundr.util.Streams

This utility class has convenience methods for dealing with Streams.

{% highlight java %} String string = Streams.readString(inputStream, "UTF-8"); byte[] data = Streams.readBytes(inputStream); InputStream resource = Streams.getResourceAsStream("classpath-resource.file"); {% endhighlight %}

com.threewks.thundr.http.Authorization

Authorization contains convenience methods for dealing with HTTP authorization policies.

com.threewks.thundr.http.ContentType

An enumeration for common content types, and utility methods for dealing with them.

com.threewks.thundr.http.Cookies

Provides a builder for creating Cookies, and some convenience methods for extracting them from a HttpServletRequest.

com.threewks.thundr.http.Header

Header contains a range of constants and convenience methods for dealing with HTTP headers.

com.threewks.thundr.http.StatusCode

An enumeration for Http status codes, and utility methods for dealing with and understanding them.

com.threewks.thundr.http.URLEncoder

Provides convenience methods for dealing with url components that need encoding.

For example components in url paths require different encoding to components in query parameters.
